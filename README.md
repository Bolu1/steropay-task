
## Description

Task for a backend role at steropay written with NodeJS(NestJS) and Prisma ORM and MySQL

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Creating a migration

```bash
# Push
$ npx prisma db push --preview-feature

# Migrate
$ npx prisma migrate dev

```
